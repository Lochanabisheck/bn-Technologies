# BN Technologies website

Responsive static website for a Dubai-based cryptocurrency, blockchain, digital asset, and Web3 education consultancy.

## Included

- Modern Dubai-inspired responsive design
- Cryptocurrency and blockchain education services
- Digital wallet and crypto-safety guidance
- Web3 and business consultancy content
- Appointment scheduling with weekday validation
- Web3Forms API email delivery
- GitHub Pages custom-domain files
- SEO metadata, sitemap, robots file, and custom 404 page

## Required before launch

1. Add the Web3Forms access key by following `WEB3FORMS-SETUP.md`.
2. Upload the package to GitHub Pages.
3. Connect `bntechnologies.ai` using `GITHUB-GODADDY-DEPLOYMENT.md`.

## Local preview

Run a static web server in this folder:

```bash
python3 -m http.server 8080
```

Open `http://127.0.0.1:8080`.
